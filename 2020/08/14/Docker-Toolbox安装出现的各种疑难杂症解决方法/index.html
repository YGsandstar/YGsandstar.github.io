sandstar
2020-08-14
Docker, 安装教程

Preface

笔者win10系统是家庭版，所以不能下载 Docker for Windows 直装，所以，如果你是家庭版，也可选择去 tb 买专业版的激活码，不过五块钱的事，当然你面临的问题就得去找专门的 Docker for Windows 的博客来解决，本文并不适用，所以请确认好你是 win10 家庭版的用户，又在 Quickstart 时遇到了一系列问题，那么希望本文会对你有所帮助。

方法一

Quickstart 有如下信息：

可能你是这种情况，先安装了 Docker for Windows 按照某教程把 Hyper-V 开启了，因为 win10 家庭版是没有 Hyper-V 的，你可能因为 Docker for Windows 把 Hyper-V 下载了，在结果不对转而安装 Docker Toolbox 时出现了问题，所以请你关闭该功能：

系统会提示你重启，但如果你重启以后，发现还是报错，可以考虑以管理员身份运行cmd，关掉该功能：

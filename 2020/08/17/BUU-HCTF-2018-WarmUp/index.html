网页源码有提示 source.php ，访问获得源码如下：

hint.php 访问得到提示 flag not here, and flag in ffffllllaaaagggg

简而言之，思路是绕过四个 if 判断，通过文件包含读取 ffffllllaaaagggg

$REQUEST 可以获取以POST方法和GET方法提交的数据，接收 file 这个参数。

file 有三个限制：

  1. 非空。
  2. 是字符串。
  3. emmm 类中的静态方法 checkFile() 要 return true

接下来看 emmm 类中的静态方法 checkFile() 四个 if 判断（相应的函数解释见代码注释）：

  1. page 也就是我们传进去的 file ，非空且为字符串。
  2. 在白名单中。
  3. 利用 mb_substr 截取 page 中第一个 ‘?’ 前面的部分，这部分需要在白名单中。
  4. url 解码，再次利用 mb_substr 截取 page 中第一个 ‘?’ 前面的部分，这部分需要在白名单中。

这里有意思的是，它并不要求你全部 if 语句都为 true ，只要在 return true 在 return false 之前，满足一个 return true 即可。

当然第二个 if 判断我们是做不到读取 flag 的同时满足在白名单中。

所以最后，我们 file 传参的限制条件只有下面这些：

  1. 非空且为字符串。
  2. 3 或者 4 判断为真。
